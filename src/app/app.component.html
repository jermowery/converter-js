Bookmarks conversion tool

This tool will convert a bookmarks file from SmartAudio Books to an Audacity Label Track.

To get your SmartAudio books bookmarks file browse to the directory on your android device using a file manager app. Go to Internal Storage -> The "Root folder" as defined in your SmartAudio Books settings -> alongside your audio files.

To import the files you get from this tool please see the Audacity Manual

Looking for the command line tool? Find it here.
